TikTok Clone App (Expo + React Native)

A modern TikTok-style mobile application built with React Native and Expo Router, featuring video feeds, user profiles, authentication, and scalable state management.

Features
🎥 Video Feed (TikTok-style scrolling)
👤 User Profiles with posts/videos
🔐 Authentication
📦 API handling with Axios
⚡ Fast lists using FlashList
🎯 Form validation with React Hook Form + Yup
📱 Cross-platform (Android, iOS, Web)
🎬 Video playback support
🧠 State management with Redux Toolkit & Zustand
🔄 Data fetching with React Query
